# AutoCuba - Dealer de Autos Premium

Una aplicación moderna de dealer de autos para Cuba con opciones de stock local e importación desde USA.

## 🚀 Tecnologías

- **Next.js 14** - Framework de React
- **Bun** - Runtime y package manager ultrarrápido
- **TypeScript** - Tipado estático
- **Tailwind CSS v4** - Estilos utilitarios
- **Radix UI** - Componentes accesibles
- **Lucide React** - Iconos

## 📦 Instalación con Bun

\`\`\`bash
# Instalar dependencias
bun install

# Ejecutar en desarrollo
bun dev

# Construir para producción
bun run build

# Ejecutar en producción
bun start
\`\`\`

## 🎨 Características

- ✅ Catálogo de vehículos con filtros avanzados
- ✅ Sistema de cotización y financiamiento
- ✅ Importación desde USA
- ✅ Modo oscuro/claro
- ✅ Diseño responsive
- ✅ Animaciones suaves
- ✅ SEO optimizado

## 🛠️ Comandos Útiles

\`\`\`bash
# Limpiar cache y dependencias
bun run clean

# Instalar nueva dependencia
bun add <package>

# Instalar dependencia de desarrollo
bun add -d <package>

# Ejecutar linting
bun run lint
\`\`\`
